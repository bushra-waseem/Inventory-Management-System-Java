# Inventory Management System (Java GUI) 📦

A robust Java-based Desktop Application that manages product inventory using a **Stack (LIFO)** data structure. This project demonstrates Graphical User Interface (GUI) design and core Data Structure concepts.

---

## 📸 Project Showcase

### 1. Main Interface
The primary dashboard where users can add items (Push) and view the current inventory list.
![Main Interface](screenshots/01%20main%20interface.png)

### 2. Remove Item (Pop)
Demonstrating the Stack's "Pop" operation, where the most recently added item is removed first.
![Pop Item](screenshots/02%20pop%20item.png)

### 3. View Top Item (Peek)
The "Peek" functionality allows users to see the top-most item in the inventory without removing it.
![Peek Item](screenshots/03%20peek%20item.png)

### 4. Error Handling
The system includes validation to ensure only valid numeric data is entered for quantity and price.
![Error Detection](screenshots/04%20error%20detect.png)

---

## 🚀 Key Features
* **Stack Data Structure:** Implements Push, Pop, and Peek operations for inventory management.
* **Dynamic GUI:** Built using Java Swing/AWT for a responsive user experience.
* **Real-time Status:** Provides instant feedback on successful operations or errors.
* **Item Tracking:** Tracks Item Name, Quantity, and Price efficiently.

## 🛠️ Technical Stack
* **Language:** Java (JDK 24)
* **UI Framework:** Swing / AWT
* **Data Structure:** Custom Stack implementation
